## 它能干什么？

**1. 快速预览 SVG**

把 SVG 文件拖进去，马上能看到图标长什么样，还能看到尺寸和大小数据，不用打开浏览器、不用开 Finder 预览，省去好多步骤。

**2. 一键优化压缩**

Trym 会自动清理 SVG 文件里那些没用的数据——多余的注释、冗余的路径、编辑器自动塞进去的各种服务信息。根据不同的图标复杂度，优化效果从 2% 到 30% 不等，虽然听起来不大，但对于一个有几百个图标的项目来说，加起来相当可观。

**3. SVG 一键转 PNG**

这个功能用起来是真的爽。有时候需要把矢量图交给运营或者客户，对方不一定能用 SVG，Trym 支持直接导出为 PNG 格式，还能适配 Web、iOS 和 Android 不同尺寸需求，一次搞定，不用再开 PS 或者 Illustrator 转一圈。

**4. 自动切换亮/暗主题**

Trym 会根据你打开的 SVG 图标颜色，自动在亮色和暗色界面之间切换——浅色图标配深色背景，深色图标配亮色背景，让预览效果更直观。这个小细节，用过的人都说贴心。
